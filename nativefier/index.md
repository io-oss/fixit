# Nativefier


找到了一个把网站转为本地APP的工具`nativefier`。

<!--more-->

## 项目地址

https://github.com/nativefier/nativefier

## 安装

1. 安装node.js
2. npm安装nativefier

```zsh
npm install nativefier -g 
```

## 使用

```
nativefier --name "MSDN" "https://docs.microsoft.com/zh-cn/"
```

{{<admonition tip "注意">}}
--name 参数用于指定生成应用的名称，但是如果是中文名，需要先生成应用，然后手动改名。
{{</admonition>}}
